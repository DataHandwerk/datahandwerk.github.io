repo.usp_persistence_set - P

type: P ( stored procedure ), modify_date: 2021-12-16 11:37:50

RepoObject_guid: BA90291C-9D61-EB11-84DC-A81E8446D5B0

Description

  • create or update RepoObject in repo.RepoObject - U for a new persistence target (table or view), based on a given persistence source (view or table)

  • create or update entries in repo.RepoObject_persistence - U

    • default properties are used, defined in this table

      • [is_persistence_truncate] = 1

      • [is_persistence_insert] = 1

      • all others bit values = 0

see details for usage in Persistence generator
How does it work:

  • insert or update repo.RepoObject_persistence - U

    • update existing RepoObject which should be a table and will be marked as persistence

    • create new RepoObject which will be a table and will be marked as persistence

  • persistence source

    • uses @source_RepoObject_guid, if not empty

    • tries to get @source_RepoObject_guid from @source_fullname

  • persistence target

    • with @persistence_RepoObject_guid an existing table can be defined as target

    • otherwise defaults are used

      • same schema as persistence source

      • if the @persistence_table_name is NULL, defaults are used:

        • name of persistence source + suffix (FROM [repo].[Parameter] WHERE [Parameter_name] = 'persistence_name_suffix')

  • persistence source NULL, persistence target NULL

    • ⇒ error
pesistence procedure naming

the default name for the persistence procedure is

  • 'persistence target schema'.'usp_PERSIST_' + 'persistence target name'

    [SchemaName].[usp_PERSIST_SourceViewName_T]

after executing repo.usp_persistence_set - P you should

  • EXEC repo.usp_main - P

  • check and update attributes in repo.RepoObject_persistence - U

  • physically create the persistence table (the procedure repo.usp_persistence_set - P will only create the code)

    SELECT
    [RepoObject_guid]
  , [DbmlTable]
  , [RepoObject_fullname]
  , [SqlCreateTable]
  , [ConList]
  , [persistence_source_RepoObject_fullname]
  , [persistence_source_RepoObject_guid]
  , [persistence_source_SysObject_fullname]
FROM
    [repo].[RepoObject_SqlCreateTable]
WHERE
    NOT [persistence_source_RepoObject_fullname] IS NULL
ORDER BY
    [RepoObject_fullname];

    • Use the sql statement in column [SqlCreateTable] to create the table

  • get the usp code in uspgenerator.GeneratorUsp_SqlUsp - V and execute it to create the persistence procedure
